# The Button

An es7 friendly Example React / Koa app

Install all dependencies:
```
npm install
```

Create a '.env' file in the project's root folder with the following information:  
```
username={Your computer's user name}
password=postgres
dbname=postgres
PORT=9000
```

## Running the app
#### Run tests  
```
npm test
```

#### Start the button
```
npm start
```

#### Debug
```
npm run debug
```
